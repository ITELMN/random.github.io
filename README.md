## 项目介绍

班级随机座位器是一个专为教育工作者设计的网页应用。它能够帮助教师快速、公平地为学生分配座位，提高课堂管理效率。

### 主要特性

- 随机分配座位
- 支持自定义座位数量
- 可上传或手动输入学生名单
- 实时预览座位安排
- 保存和加载座位配置
- 导出座位表为Excel文件
